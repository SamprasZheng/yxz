# scripts/video-drafts

Folder for intermediate and final artifacts of the video pipeline. All files are named using the blog slug as the base; each step in the pipeline reads the previous step's artifact and adds new fields. These are not source-of-truth files — they can be safely deleted and the pipeline re-run.

| File | Generated by | Purpose |
|---|---|---|
| `<slug>.json` | `blog-to-script.cjs` | TLDR narration script, chapters, blog path; later steps append fields such as `mp4Path` and `youtubeId` |
| `<slug>.mp3` | `tts.py` / `script-to-audio.cjs` | TTS-synthesized audio file (edge-tts zh-TW) |
| `<slug>.srt` | `tts.py` / `script-to-audio.cjs` | Chinese subtitle file aligned with the mp3 |
| `<slug>.mp4` | `audio-to-video.cjs` | 1920×1080 H.264/AAC finished video; input for `youtube-upload.cjs` |

The `debug.*` files currently committed in the repository are example artifacts from `blog/2025-06-02-debug-playbook.md`.

For the pipeline flow and the responsibilities of each script, see [`../README.md`](../README.md).
