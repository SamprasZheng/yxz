# 1️⃣ Fundamentals — What Is Polkadot

**→ Read the main article: [Polkadot — The Next-Generation Blockchain](/blog/lop)**

## Key Takeaways

1. Polkadot is not a single chain — it is a **multi-chain protocol** consisting of a Relay Chain plus multiple Parachains.
2. The cross-chain messaging protocol **XCM** enables assets and messages to move trustlessly between chains.
3. **OpenGov** is Polkadot's on-chain democracy system; all protocol upgrades are passed by vote of DOT holders.
4. Three highlights of Polkadot 2.0: Agile Coretime, Elastic Scaling, and Async Backing.
