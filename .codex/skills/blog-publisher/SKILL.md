# Blog Publisher

## Draft From Wiki

Use this when a wiki synthesis page is ready to become a post:

```bash
cd my-website
yarn draft:from-wiki ../wiki/synthesis/<page>.md
```

The generated draft lands in `my-website/draft/` with `draft: true`. Review claims, citations, and tone before moving it into `my-website/blog/`.

## Blog Frontmatter

Use valid authors from `blog/authors.yml` and valid tags from `blog/tags.yml`. The `polkadot` tag maps to `/blog/tags/dot`; keep navbar links aligned with that route.

Do not hand-edit `blog/live-*.md`; those files are generated by `yarn generate:living-topics`.

## Publishing Checks

From `my-website/`, run targeted checks:

```bash
yarn validate:og
yarn typecheck
yarn build
```

Run `yarn verify:quality` before publishing broad content batches.

## Video Pipeline

For posts marked `video_ready: true`, follow `.claude/commands/auto-publish.md`. Pause for script approval before TTS, video rendering, YouTube upload, embed, and deploy.
