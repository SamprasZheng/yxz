# Anthropic

AI safety company and developer of the Claude family of LLMs. Founded 2021 by Dario Amodei, Daniela Amodei, and others who departed OpenAI.

## Key facts

- **Products:** Claude chatbot (claude.ai); Claude API; Claude Code CLI; Agent SDK
- **Models (as of June 2026):** Opus 4.8, Sonnet 4.6, Haiku 4.5 — tiered for reasoning, coding, and speed
- **Revenue:** ~$47B annualised (June 2026)
- **IPO:** Confidential S-1 filed with SEC June 1, 2026; ~$965B valuation; potential $1T debut would rank second- or third-largest IPO ever
- **Investors:** Google, Amazon, Spark Capital, and others
- **Compute deal:** Partnership with SpaceX announced alongside higher usage limits (June 2026)

## Claude Code

Claude Code is Anthropic's agentic CLI for software engineering, running within the Claude Agent SDK. Project "Marlin" (via Snorkel AI) uses contractors at up to $280/task to fine-tune it. Starting June 15, 2026, programmatic usage via subscription plans moves to a separate monthly credit pool.
