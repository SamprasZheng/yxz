# Anthropic

AI safety company (San Francisco) founded 2021 by former OpenAI researchers including Dario Amodei and Daniela Amodei; creator of the Claude model family (Haiku / Sonnet / Opus) and Claude Code, the agentic coding CLI. Operates under a Public Benefit Corporation + Long-Term Benefit Trust governance structure.

**Key products:** Claude (chat + API), Claude Code (agentic CLI + IDE extension), Workspaces (enterprise), Amazon Bedrock and Vertex AI integrations, Agent SDK.

**June 2026 milestones:** Filed confidential SEC paperwork for an IPO at a likely valuation >$1 trillion (June 1); launched Claude Opus 4.8 with Dynamic Workflows (parallel multi-agent orchestration for large-scale coding tasks); subscription plans split into a separate agentic credit pool from June 15; Claude AI experienced a service outage June 2; compute partnership with SpaceX announced.
