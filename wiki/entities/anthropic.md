# Anthropic

US AI safety company; maker of Claude LLM family (Haiku / Sonnet / Opus / Fable / Mythos); founded 2021 by Dario Amodei, Daniela Amodei, and colleagues from OpenAI; $965B valuation (July 2026, surpassing OpenAI's $852B); annualized revenue pace ~$47B (May 2026), expecting profitability 2029.

Key products: Claude API, Claude.ai, Claude Code (CLI/IDE agentic coding tool), Claude Science (drug discovery workbench, beta), Claude for Business/Enterprise.

Notable 2026 events: Fable 5 + Mythos 5 export-control pause (June 12) and full restore (July 1); Claude Sonnet 5 launched June 30 as new default model (63.2% agentic coding benchmark); California statewide deal at 50% discount; Alibaba announced ban of Claude Code effective July 10 over alleged anti-abuse mechanism.
