# Anthropic

US AI safety company founded 2021 by Dario Amodei, Daniela Amodei, and other ex-OpenAI researchers. Builds the Claude family of LLMs (Claude 3/4/5 families; Haiku / Sonnet / Opus / Fable model tiers). Developer platform: Claude API, Claude Code CLI, Model Context Protocol (MCP).

**2026 notable moves:**
- "Ode with Anthropic" — $1.5B AI implementation JV with Blackstone, Hellman & Friedman, and Goldman Sachs (confirmed mid-July 2026).
- Claude for Teachers: free premium Claude for verified US K-12 educators (July 2026).
- SpaceX compute deal: higher usage limits tied to new compute capacity (2026).
- Claude Code: added background sessions, admin console insights, device-verification for remote steering.
