# Anthropic

AI safety company founded 2021 by Dario Amodei, Daniela Amodei, and others (ex-OpenAI). Builder of the Claude model family (Haiku / Sonnet / Opus) and the Claude Code agentic development tool. Filed a confidential Form S-1 with the SEC on June 1, 2026 for a proposed IPO led by Morgan Stanley and Goldman Sachs.

## Key products

- **Claude** — frontier LLM family; latest release Claude Opus 4.8 (June 2026) adds effort control and dynamic workflows.
- **Claude Code** — terminal/IDE/web agentic coding assistant; currently spans terminal, IDE, web, desktop, and scheduled agent workflows.
- **Project Glasswing / Claude Security** — cybersecurity-focused Claude deployment; expanded to ~150 orgs June 2026.
- **Claude Partner Network** — Services Track and Partner Hub launched June 3, 2026.
