# Anthropic

AI safety company founded 2021 by Dario Amodei, Daniela Amodei, and other ex-OpenAI researchers. Primary product is the Claude family of LLMs (Haiku / Sonnet / Opus). Flagship developer tool is [[concepts/claude-code]].

## Key facts (as of 2026-05-29)

- **Valuation**: $965B (May 2026 round); raised $65B led by Altimeter Capital, Dragoneer, Greenoaks, and Sequoia Capital — eclipsing OpenAI's valuation for the first time.
- **Revenue run rate**: $47B annualised (May 2026); Q1 2026 usage grew 80× year-on-year.
- **Latest model**: Claude Opus 4.8 (1M-token context, 128k max output; available on Claude API, Amazon Bedrock, Vertex AI).
- **Developer platform**: Claude Code CLI + Claude Managed Agents (self-hosted sandbox + private MCP server support as of May 2026).
- **Investors**: Amazon, Google (Alphabet), Altimeter Capital, Spark Capital, Dragoneer, Greenoaks, Sequoia.
- **Notable partnerships**: SpaceX (Colossus data centre capacity), Gates Foundation ($200M).
