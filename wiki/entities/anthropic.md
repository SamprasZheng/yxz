# Anthropic

AI safety company founded 2021 by Dario Amodei, Daniela Amodei, and Christopher Olah (among others) after departing OpenAI. Builds the Claude LLM family (Opus / Sonnet / Haiku tiers) and the Claude Code developer platform.

## Key facts

- **Revenue**: 80× annualised Q1 2026 growth (Dario Amodei, Code with Claude 2026)
- **Ranking**: #1 CNBC Disruptor 50 (2026), overtaking OpenAI
- **Compute**: $1.25 B/month deal with xAI Colossus data centre (Memphis, TN) through May 2029
- **Products**: Claude API, Claude Code (CLI + web + IDE), Claude Managed Agents (public beta 2026-05-19), Claude Security (Project Glasswing, public beta)
- **Safety focus**: Interpretability research led by Christopher Olah; Responsible Scaling Policy; Constitutional AI methodology
