# Anthropic

US AI safety company and frontier model lab; founded 2021 by Dario Amodei, Daniela Amodei, and other ex-OpenAI researchers. Developer of the Claude model family (Haiku / Sonnet / Opus / Fable / Mythos tiers) and the Claude Code agentic development environment.

## Key facts

- Headquarters: San Francisco, CA
- Filed confidential S-1 with the SEC on June 1, 2026 — IPO process initiated; targeting public markets
- Claude model family: Haiku (fast/cheap) → Sonnet → Opus → Fable → Mythos (new top tier, June 2026)
- Claude Code: agentic software development CLI; billing split into interactive (subscription) and headless/API (API-rate credit) effective June 15, 2026
- Project Glasswing: collaborative software security initiative; 150+ partner organizations; 10,000+ high/critical flaws found in initial cohort (June 2026)
- Claude Partner Network launched June 2026 with TCS, DXC for regulated industries
