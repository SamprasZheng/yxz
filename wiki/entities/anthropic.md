# Anthropic

US AI safety company; creator of the Claude LLM family. Founded 2021 by Dario Amodei, Daniela Amodei, and a team from OpenAI. Headquartered in San Francisco.

## Key facts (as of 2026-06-06)

- Filed confidential S-1 for IPO on June 1, 2026; valuation ~$965B; May 2026 revenue run-rate ~$47B (up from ~$10B the prior year).
- Claude Opus 4.8 is the current flagship model (default on Max, Team Premium, Enterprise pay-as-you-go, and API as of June 2026).
- Claude Code writes >80% of Anthropic's own merged production code as of June 2026.
- Starting June 15, 2026, agentic/programmatic workloads move to a separate monthly credit pool rather than drawing from interactive-chat subscriptions.

## Products

- **Claude** — chat + API; Haiku / Sonnet / Opus model tiers
- **Claude Code** — agentic coding system; reads codebases, edits files, runs tests, commits code
- **Project Glasswing / Claude Mythos Preview** — AI-native cybersecurity scanning; ~150 partner organizations as of June 2026; found >10,000 high/critical vulnerabilities in initial cohort
- **Claude Partner Network** — Services Track + Partner Hub launched June 3, 2026
