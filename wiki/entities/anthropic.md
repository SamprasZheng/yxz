# Anthropic

US AI safety company founded 2021 by Dario Amodei, Daniela Amodei, and colleagues who departed OpenAI. Develops the Claude family of LLMs and the Claude Code developer tooling. Operates Claude API (Anthropic API), Claude Partner Network, and Project Glasswing (AI-safety critical-infrastructure partnerships). As of June 2026, the company filed a confidential S-1 with the SEC (June 1, 2026) for an IPO, with Morgan Stanley and Goldman Sachs as lead underwriters; the $65B Series H lifted the pre-IPO valuation to ~$965B. On June 5, 2026, Anthropic publicly called for a "coordinated and verifiable pause" in AI development, citing the pace of improvement as a risk to meaningful human control.

## Key products

- Claude (Sonnet, Opus, Haiku model families)
- Claude Code (CLI + IDE + web agentic coding tool)
- Claude Agent SDK (headless/scheduled agent workloads)
- Project Glasswing (critical infrastructure AI safety program)
- Claude Partner Network (Services Track + Partner Hub, June 2026)
