# Anthropic

AI safety company founded 2021 by Dario Amodei, Daniela Amodei, Tom Brown, and others; creator of the Claude LLM family. As of June 2026: $965 B valuation (Series H, $65 B raised); confidential S-1 draft filed with the SEC June 1, 2026; racing [[entities/openai]] to a fall 2026 public listing.

## Key Products (2026)

- **Claude family** — Haiku 4.5 / Sonnet 4.6 / Opus 4.8; **Fable 5** (Mythos-class, public, launched June 9 2026); Mythos 5 (Project Glasswing restricted)
- **Claude Code** — CLI + IDE agent SDK; agentic coding, MCP integration, background sessions, post-session hooks
- **Claude Developer Platform** — API, Managed Agents scheduled deployments, vault credentials, session thread webhooks
- **Code with Claude** — developer conference (May 2026); proactive workflows, capability curve unveiled

## Notable Facts (2026-06)

- Claude Fable 5: SWE-Bench Pro **80.3%** (vs Opus 4.8 69.2%, GPT-5.5 58.6%); first model to break **90%** on complex analytics benchmarks; $10/M input tokens, $50/M output tokens; 1 M token context
- Confidential S-1 filed with SEC June 1, 2026; IPO race with [[entities/openai]] (which filed June 8)
- Compute partnership with SpaceX for Fable 5 inference scaling
- Claude Sonnet 4 + Opus 4 retiring from API June 15, 2026; migration path to Sonnet 4.6 / Opus 4.8
