# Anthropic

US AI safety company founded 2021 by Dario Amodei, Daniela Amodei, and other OpenAI alumni. Creator of the **Claude** model family (Haiku / Sonnet / Opus; generations 3 → 4 → 5) and **Claude Code** (agentic coding CLI with Cowork, Remote Control, and IDE plugins). Holds FedRAMP High authorization for Claude for Government Desktop. Anthropic's stated core thesis is "responsible development toward safe AGI."

## Key products (as of 2026-07)

- Claude model family — Haiku 4.5 / Sonnet 4.6 / Sonnet 5 (default in Claude Code July 2026, 1M-token context) / Opus 4.8
- Claude Code — agentic CLI; Cowork moving to cloud multi-device (July 2026); FedRAMP High Government Desktop public beta
- Claude API (Anthropic Workbench, third-party integrations including NemoClaw/OpenClaw ClawRouter)

## Notable events

- 2026-07-08: China MIIT advisory warns of "security backdoor" in Claude Code versions 2.1.91–2.1.196 (location-tracking for anti-distillation enforcement)
- 2026-07-10: Alibaba bans employee use of Claude Code; Anthropic confirms tracking code, says China-based users were never authorized
- 2026-07: Claude Cowork moves to cloud; Government Desktop FedRAMP High public beta; Sonnet 5 becomes default model
