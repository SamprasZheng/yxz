# Anthropic

Anthropic PBC is an AI safety company and the creator of the Claude family of large language models, headquartered in San Francisco.

## Key facts

- **Founded**: 2021 by Dario Amodei, Daniela Amodei, and other former OpenAI researchers
- **Products**: Claude model family (Haiku / Sonnet / Opus), Claude.ai, Claude Code CLI, Claude API, Agent SDK
- **Revenue**: $4.8B Q1 2026; Q2 2026 projection $10.9B; first quarterly operating profit $559M Q1 2026; annualized run-rate > $30B
- **Funding**: Series H (2026) — $65B raised, ~$965B post-money valuation; IPO expected fall 2026
- **Compute**: Amazon partnership for up to 5 GW of Trainium2/3 capacity; ~1 GW online by end of 2026
- **Policy (June 2026)**: Third-party agents (OpenClaw, etc.) regain Claude subscription-tier access under monthly usage limits effective June 2026; automated workload billing (Agent SDK / `claude -p` / GitHub Actions) moves to a separate monthly credit pool at full API rates effective June 15, 2026
