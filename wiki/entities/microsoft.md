# Microsoft

Microsoft Corp is a US technology company; in the AI-agent era its most relevant products are GitHub Copilot, Microsoft 365 Copilot, Azure AI Foundry, and Windows AI APIs.

## Key facts

- **Headquarters**: Redmond, WA
- **AI agent products (June 2026)**: Windows Agent Framework APIs; Copilot agent mode; Windows Agent Store (announced Microsoft Build June 2, 2026); Microsoft Agent 365 (GA); M365 Copilot (enterprise agent layer)
- **Claude Code decision**: Microsoft cancelled Claude Code licenses for thousands of internal engineers by June 30, 2026, switching to GitHub Copilot CLI after usage costs exceeded the annual AI budget
- **OpenClaw-like agent**: Microsoft is building an OpenClaw-inspired agent inside M365 Copilot for enterprise; expected reveal at Build June 2, 2026
- **Azure AI Foundry**: Azure's hosted agent platform powering Copilot and third-party agent orchestration
