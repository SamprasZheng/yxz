# OpenAI

US AI research company; develops the GPT and ChatGPT product line, Codex (autonomous coding agent), DALL-E, Whisper, and the OpenAI API. Co-developed the Agent Commerce Protocol (ACP) with Stripe. Key 2026 milestones: GPT-5.5 series (including GPT-Rosalind for life sciences); Codex expanded to all roles June 3, 2026; ChatGPT memory "dreaming" system for auto-updating context; Workspace Agents for Enterprise/EDU (free preview through July 6, then credit-based); gpt-oss-120b/20b Apache 2.0 open weights released August 2025. OpenAI frontier models and Codex became available on AWS marketplace June 1, 2026.

## Key products

- ChatGPT (consumer + Enterprise + EDU tiers)
- Codex (autonomous coding; expanding to all roles June 2026)
- GPT-Rosalind (life sciences research: medicinal chemistry + genomics)
- OpenAI API (GPT-5.5, gpt-oss open weights)
- Workspace Agents (team-shared multi-agent workflows)
