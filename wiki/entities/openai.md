# OpenAI

US AI lab founded 2015, transitioned to "capped-profit" model 2019. Creator of the GPT model family, ChatGPT, Codex, DALL-E, Whisper, and the OpenAI API. Backed by Microsoft ($13B+). As of July 2026 the latest model is **GPT-5.6** (Sol/Terra/Luna, released July 9 after a Trump-administration-requested public-rollout delay). GPT-5.6 ships a built-in router, which is noted in [[concepts/tiered-inference]] as a productization signal for cascaded inference.

## Key products (as of 2026-07)

- GPT-5.6 — Sol (most capable) / Terra (balanced) / Luna (fast); GA July 7–14, 2026
- ChatGPT Work — workplace agent for document/spreadsheet/presentation generation from connected apps
- ChatGPT Agent Mode v2 — persistent memory, scheduled tasks, GitHub/Jira connectors (Plus/Team)
- Codex — code generation and agentic dev CLI; center of Build Week July 13–21, 2026
- OpenAI API / Responses API — Programmatic Tool Calling (ZDR-compatible), Multi-agent beta

## Notable events

- 2026-07-07: GPT-5.6 GA for Enterprise API
- 2026-07-09: Public release of GPT-5.6; ChatGPT Work launched; delayed at Trump administration request
- 2026-07-13: Build Week opens (global developer event centered on Codex; runs July 13–21)
- 2026-07-14: Full Team rollout of GPT-5.6
