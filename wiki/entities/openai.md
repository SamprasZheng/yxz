# OpenAI

US AI company, maker of the GPT / ChatGPT / Codex / DALL-E family; originally a non-profit (2015), restructured as a capped-profit entity. Founded by Sam Altman, Greg Brockman, Ilya Sutskever, Elon Musk, and others.

## Model Family (2026)

- **GPT-5.5** (June 2026) — current flagship; replaced GPT-5.2 across all ChatGPT tiers 2026-06-12
- **gpt-oss-120b / gpt-oss-20b** — open-weight release (Apache 2.0, Aug 2025); open-as-funnel strategy
- Codex family for code generation / agentic coding tasks

## Key Developments (2026)

- Confidential draft S-1 filed with the SEC 2026-06-08 (one week after Anthropic's filing)
- Acquired Ona 2026-06-11 (details sparse)
- Major ChatGPT memory upgrade (June 2026)
- OpenAI Economic Research Exchange launched (external research collaboration on AI economic impacts)
- OpenAI Library launched for Enterprise / Edu / Healthcare workspaces
