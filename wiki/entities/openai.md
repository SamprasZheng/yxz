# OpenAI

AI lab founded 2015; creator of the GPT model family, ChatGPT, Codex, Sora, and the o-series reasoning models. Pursuing an IPO following Anthropic's June 1, 2026 S-1 filing. Peter Steinberger (founder of [[concepts/openclaw]]) joined OpenAI in February 2026 to lead personal agents.

## Key products (June 2026 snapshot)

- **ChatGPT / GPT family** — consumer and API LLM products; GPT-Rosalind variant for life sciences launched June 2026.
- **Codex** — code-generation agent; expanded with Computer Use on Windows June 2026.
- **ChatGPT Sites** — build-and-deploy internal web apps via Codex with hosted URLs (in preview June 2026).
- **OpenAI Agent Platform** — personal-agents initiative; Steinberger hired to lead.
