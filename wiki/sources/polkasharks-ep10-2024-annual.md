# 波卡解碼EP10: 波卡2024年報

**Author:** [[entities/polkasharks]] | **Date:** 2024-12-30 | **Series:** 波卡解碼

## Summary

Comprehensive 2024 annual review of Polkadot. Covers security, performance, governance, and ecosystem milestones. Includes 2025 roadmap.

## Key claims

### Security & Decentralization
- Active validator set grew to 500 — cited as highest decentralization degree
- 52.5%+ of DOT staked; total value ~$8 billion USD
- Inflation model changed from exponential to linear; year-one rate: 8%

### Performance
- "Spammening" stress test achieved 143,000+ TPS throughput
- Mythos platform: 3 million transactions in a single day (September 2024)
- Async Backing + Agile Coretime + Elastic Scaling shipped; Rollup block time cut from 12s → 6s

### Ecosystem
- [[entities/mythical-games]] migrated from Ethereum to Polkadot
- Mandala launched Indonesia-focused Rollup
- Centrifuge expanded Real World Asset (RWA) tokenization

### Governance
- 950+ referenda initiated since early 2024; ~77% are treasury-related proposals
- Treasury spending grew 3.2× vs. 2023

### 2025 Roadmap
- Asset Hub smart contracts
- Polkadot Virtual Machine (PVM)
- XCM V5
- JAM mainnet (target ~2026)
